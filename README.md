# Songkran
## Thing needed
* Node.js
* Angular 7
    ```bash
    npm install -g @angular/cli
    ```
* json-server
    ```bash
    npm install -g json-server
    ```
## How to run
1. run json server first
	```bash
	json-server --watch db.json
	```
2. run Angular server
	```bash
	ng serve --open
	```
3. Angular will open browser at http://localhost:4200/
